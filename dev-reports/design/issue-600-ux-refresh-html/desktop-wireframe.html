Marketing cut points
1. Hero count-up Running / Waiting / Completed の数が増減する
2. Waiting card opens 「人の一言で再開できる」価値が見える
3. Running bars move 複数AIが同時に進む様子を短尺で表現
4. Feed updates ブラウザを見ていない間も進捗が蓄積される
Wireframe notes
Home is not admin Repository管理やExternal Appsは別導線へ
Needs Attention first Waiting / Error / Done-unreviewed を優先
Sessions becomes secondary 全件探索は専用画面に譲る
