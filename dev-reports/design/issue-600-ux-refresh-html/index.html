CommandMate v0.5 UX Wireframes

ホーム画面を「管理一覧」から「AI開発の管制塔」へ再設計するためのワイヤーフレームです。 PC版は俯瞰と采配、モバイル版は即判断と即返信を主目的にしています。

Desktop

PC版ホーム wireframe

役割は Mission Control。複数セッションの状態、要返信、直近の成果を 一画面で判断できる構成です。

  • 上段で「今どうなっているか」を数値とライブ状態で即把握
  • 中央で「人が介入すべきもの」を優先表示
  • 下段で一覧確認や探索へ自然に接続
  • 動画では状態変化とライブ感が最も映える構成
PC版を開く
Mobile

モバイル版ホーム wireframe

役割は Inbox / Reply Center。スマホでは探索よりも、 「何に返事すればよいか」「どれが完了したか」を素早く処理できることを優先します。

  • 最上部で待ち・進行中・完了を端的に表示
  • 下に行くほど詳細になる縦導線
  • ボトムナビで親指操作を最適化
  • 短尺動画で「PCを離れても開発が進む」印象を出しやすい
モバイル版を開く
Journeys

ユーザシナリオ / 導線整理

初回導入、リポジトリ登録、同期、worktree選択、実行、レビュー、管理までを ライフサイクルで整理したUX設計ページです。

  • 全ユーザシナリオを棚卸し
  • 各シナリオの正しい入口画面を定義
  • Desktop / Mobile で同じ mental model に統一
  • 「登録は Repositories」「実行は Sessions / Detail」へ整理

Selected nav

Sessions / Repositories / Review / More

グローバルナビで各項目を選択したときに見える画面を、Desktop / Mobile 両方で 定義したワイヤーフレーム群です。

  • `Sessions` は現行メイン画面の進化版
  • `Repositories` は登録・Sync・削除・復元の専用面
  • `Review` は完了・停滞・要承認のキュー
  • `More` は設定・外部連携・アカウント管理

Desktop only

PC専用画面 + Sessions遷移

PC利用時の各ナビ選択画面を独立HTMLで切り出し、あわせて `Sessions → Worktree Detail → Prompt / Review` の遷移も整理しています。

  • `Sessions` は現行メイン画面のPC進化版
  • `Repositories` は登録と同期のPC作業面
  • `Review` は未確認成果を処理するPCキュー
  • `More` は補助設定と外部連携のPC面
  • `Sessions Flow` は画面遷移と戻り先を整理
